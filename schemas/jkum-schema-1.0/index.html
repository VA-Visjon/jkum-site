    Title: EPSG
    Type integer
    Required Yes
    Default ""

    Description: A valid EPSG code, see www.epsg.io for reference.

    Examples: 

    25832

    25833

    5110

    5111
    Restrictions
    Must match regular expression ^[0-9]{4,5}$ Test

    Type string
